# Личный финансовый планер

## Описание
Приложение для учета доходов и расходов, анализа финансовых привычек и построения графиков.  
Позволяет добавлять финансовые операции, сохранять их в CSV, анализировать данные с помощью pandas и визуализировать доходы и расходы через matplotlib.

Программа демонстрирует:  
- Работа с классами и ООП  
- Сохранение и загрузку данных в CSV  
- Анализ финансов с использованием pandas  
- Визуализацию данных (графики доходов/расходов, круговая диаграмма, топ расходов)  
- Проверку корректности пользовательского ввода через регулярные выражения  
- Графический интерфейс с использованием Tkinter

## Установка
1. Клонируйте репозиторий:
```bash
git clone https://github.com/Denny018/finance_planner.git
```
2. Перейдите в папку проекта:
```bash
cd finance_planner
```
3. Установите зависимости:
```bash
pip install pandas matplotlib
```
4. Для запуска графического интерфейса используйте:
```bash
python gui.py
```

### Основные функции программы:
- Добавление доходов и расходов через форму
- Просмотр списка операций
- Расчет текущего баланса
- Построение графиков:
  - Доходы и расходы по времени
  - Круговая диаграмма расходов по категориям
  - Топ-N категорий расходов
- Для запуска unit-тестов используйте:
```bash
python -m unittest tests/test_core.py
```
